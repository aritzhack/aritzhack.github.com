Late LudumDare 27 (LLD27 or LD27-1)

Screenshots

What is it?

'Late LudumDare 27' is a game I am making from scratch right after LD27 finished, in order learn new things and practice game-making.

It is being made using pure Java2D, without any external libraries.

Where can I get it?

'LLD27' is Open Source, and the source code can be found at GitHub

As for compiled versions, you'll have to compile it yourself, since I do not distribute any built version

Contributing
