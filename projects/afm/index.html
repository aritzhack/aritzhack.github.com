AFM

What is it?

AFM is a Minecraft mod which I am "currently developing". It is currently updated for Minecraft version 1.6.2, and although I do not distribute built copies of it, you can feel free to compile it yourself. Just make sure you satisfy the license

Where can I get it?

AFM is Open Source, and the source code can be found at GitHub
As for compiled versions, you'll have to compile it yourself

Contributing

You can contribute to the mod by:

  • Leaving an issue describing the problem, and letting me solve it by myself (Recommended)
  • Forking the repo, making your own changes, and then making a pull request.
