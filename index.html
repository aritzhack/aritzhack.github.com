My Projects

AFM

What is it?

AFM is, as stated before, a mod which will include everything I can, using most of the Forge API's features. AFM stands for Aritzh's First Mod (Not very original, I know), however, this might be a temporary name.

Why make a Omni-mod (Mod that includes lots of things)?

Well, I'm a newbie modder, so I don't know a lot about it. That's why I try to learn and implement as many things as possible, so that when I need to add something I already added, I know how to do it. In addition, I might make some modding tutorials once AFM is "finished" (or at leas when it has a bunch of interesting features), which makes me try to write the mod as cleanly as possible, so that it is easily readable.
